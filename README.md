# selenium-phptravels-test
Automatización con Selenium – PHPTravels

Este repositorio contiene un proyecto sencillo de automatización utilizando Selenium WebDriver, desarrollado como parte de una simulación académica sobre ejecución de pruebas.
Tecnologías utilizadas
Java
Selenium WebDriver
Maven
ChromeDriver
Ejecución
Clonar el repositorio.
Abrir el proyecto en NetBeans, IntelliJ o Eclipse.
Actualizar dependencias Maven.
Ejecutar el archivo de prueba.
